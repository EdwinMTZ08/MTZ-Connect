# Laboratorios de Informática Forense y Preservación de Evidencia Digital

Este repositorio contiene la documentación, metodologías y demostraciones en video de las prácticas de informática forense realizadas en entornos controlados de Linux (Kali Linux) y Windows. El objetivo de estos laboratorios es aplicar los principios de la Guía NIST y las mejores prácticas en la adquisición y recuperación de datos de manera íntegra.

---
## 📁 Índice de Prácticas y Demostraciones en Video

### 1. Práctica de Tipos de Datos y Preservación Forense
*   **Descripción:** En este laboratorio se abordan las metodologías para la identificación y manejo de diferentes tipos de datos forenses. Se profundiza en los conceptos de adquisición segura, asegurando la creación de copias bit a bit exactas (imágenes forenses) e implementando principios de bloqueo de escritura (*Write-Blocking*) para salvaguardar la evidencia original contra cualquier alteración o sobreescritura accidental.
*   **📺 Demostración en Video:** [Ver práctica de tipos de datos forenses](https://youtu.be/AJWwegvb4C4?si=ZiM-iqP6EhjxoRfC)

### 2. Simulación Forense y Análisis de Evidencia Eliminada
*   **Descripción:** Simulación de una investigación digital utilizando la plataforma **Autopsy** en Kali Linux. El laboratorio consistió en montar e inspeccionar sectores de datos en imágenes de disco virtual raw (sistema de archivos FAT16) con el fin de rastrear, identificar y extraer artefactos ocultos y credenciales que fueron eliminadas del sistema.
*   **📺 Demostración en Video:** [Ver práctica de análisis de evidencia eliminada](https://youtu.be/N3T503tAxDI?si=iI-EJKIUkrfjrm53)

### 3. Recuperación Avanzada de Archivos (File Carving)
*   **Descripción:** Ejecución de procedimientos de recuperación de datos mediante el uso de la herramienta **PhotoRec** en un entorno Linux. Se realizó un escaneo profundo en sectores de una partición simulada para reconstruir y restaurar archivos e imágenes borradas de forma íntegra, verificando su estructura mediante la consola de comandos.
*   **📺 Demostración en Video:** [Ver práctica de recuperación de datos](https://youtu.be/J0hAA6gAUjc?si=67SQwpvOZcpsl0PS)

## 🛠️ Herramientas y Tecnologías Utilizadas
*   **Sistemas Operativos:** Kali Linux / Windows CMD
*   **Herramientas Forenses:** Autopsy Forensic Browser, PhotoRec, ExifTool
*   **Conceptos Clave:** Write-Blocking, Adquisición de imágenes raw, File Carving, Integridad de la Evidencia (Hashes MD5/SHA-256)
